利用GitHub和Hexo搭建免费静态博客

2016-07-27
  • Blog
  • Github
  • Hexo

最近突然突发奇想，想弄一个博客。原因呢也很简单，就是把自己之前做过的一些项目整理一下发上来，分享给大家。同时用自己的名字作为域名，感觉也是棒棒的呢~

由于最近工作比较忙而且还有报告要写，所以将近一周才把这个博客的架子搭出来。之前从来没有弄过网站，折腾的过程中也走了不少弯路，所以就把这个博客的诞生作为第一篇文章吧:

  • 开发环境：OS X El Capitan
  • 静态网页生成工具： Hexo
  • 托管平台： GitHub Pages
  • 域名购买： 万网

Hexo

以下内容参考自Hexo官网：

什么是 Hexo

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

安装

安装 Hexo 只需几分钟时间，若您在安装过程中遇到问题或无法找到解决方式，请提交问题，我会尽力解决您的问题。

安装前提: 安装 Hexo 相当简单。然而在安装前，您必须检查电脑中是否已安装 Node.js 和 Git
如果您的电脑中已经安装上述必备程序，那么恭喜您！接下来只需要使用 npm 即可完成 Hexo 的安装。
